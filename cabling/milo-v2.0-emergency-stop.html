Milo V2.0 Emergency Stop

Diagram

Milo V2.0 utilises 2 emergency stop circuits, implemented using 2
normally closed switches on each emergency stop button. The first
circuit is used to trigger the contactor, which cuts power to the
VFD (pins 1-4) and motors (pins 5-6). The second circuit is used
to send an emergency stop signal to the mainboard, which triggers
an emergency stop in RepRapFirmware and allows the operator to
investigate before rebooting.
