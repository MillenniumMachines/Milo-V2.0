These parts are specific to the LDO Motors Milo v2 kit that includes an aluminum spindle mount. You need to print:

**1) EITHER-**
* [a]-LDO-80mm-Spindle-Mount-Front-x1-A.stl AND Logo-insert-x1-A.stl  
**OR**
* [mc]-LDO-80mm-Spindle-Mount-Front-x1-A.step, which is a multipart object so the logo can be printed in your accent color in place  

**2) BOTH-**
* LDO-Spindle-Mount-Back-Top-x1-A.stl  
* LDO-Spindle-Mount-Back-Bottom-x1-A.stl
