Millennium Machines Milo V2 is the next evolution of our open-source desktop CNC mill, building on the OpenBuilds MiniMill concept with major upgrades in size, rigidity, and customization. Designed for DIY builders, Milo V2 is More than capable of cutting aluminum (and steel if you're smart about it) while excelling at composites, woods, and plastics.

Milo V2 features a larger work area, fully redesigned plates and HOIs, simplified tramming, and magnetic way covers for easy maintenance. Builders can choose between leadscrews or ballscrews, select from two X-axis configurations, and benefit from new auto-adjusting anti-backlash nuts for long-term accuracy.

Milo V2 uses a moving-table X-axis, while its sibling Miley V2 features a stationary X-axis—both supported by the same LDO kit.

⚠️ IMPORTANT: ELECTRONICS NOT INCLUDED ⚠️

This repository does NOT include electronics.
To build the electronics system, you must use the [Universal Electronics Box](https://github.com/MillenniumMachines/Universal-Electronics-Box) repository.
