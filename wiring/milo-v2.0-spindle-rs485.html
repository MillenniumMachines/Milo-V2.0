Milo V2.0 Spindle Power and Control (RS485 / Modbus)

Diagram

Milo V2.0 with the Scylla V1.0 mainboard can control a VFD via RS485, which
is the simplest and most reliable way for 2-way communication between the
mainboard and the VFD. This setup should be used in most cases if supported
by the firmware and the VFD.
